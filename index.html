Javascript Flashcards!

Javascript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A symbol that is used to perform an operation.

Variables

Containers for storing data values.

Variable Naming Conventions

Use camel case and an underscore between words. The function variable name should match some patterns which shows that it is a function.

Functions

A "subprogram" that can be called by code external (or internal in the case of recursion) to the function. Values can be passed to a function, and the function will return a value.

Methods

Actions that can be performed on objects and contain function definitions.

Arguments

The arguments object is an Array-like object corresponding to the arguments passed to a function.

Parameters

A parameter is a named variable passed into a function. Parameter variables are used to import arguments into functions.

Return

The return statement ends function execution and specifies a value to be returned to the function caller.

Chaining Methods

Method chaining, also known as named parameter idiom, is a common syntax for invoking multiple method calls in object-oriented programming languages.

Strings

The String global object is a constructor for strings, or a sequence of characters.

Booleans

In computer science, a boolean is a logical data type that can have only the values true or false. A boolean is how a programming language lets you represent true and false.

Undefined

The global undefined property represents the primitive value undefined. It is one of JavaScript's primitive types.

NaN

The global NaN property is a value representing Not-A-Number.

Escape

The deprecated escape() function computes a new string in which certain characters have been replaced by a hexadecimal escape sequence.

Alert

method displays an alert dialog with the optional specified content.

Comments

add explanatory notes to the code or to prevent the browser from interpreting specific parts of the style sheet. By design, comments have no effect on the layout of a document

jQuery

JavaScript Library that focuses on simplifying DOM manipulation, AJAX calls, and Event handling. It is used by JavaScript developers frequently.

Attributes

extends a tag, changing its behavior or providing metadata. An attribute always has the form name=value (the attribute's identifier followed by its associated value).

Loops

Loops offer a quick and easy way to do something repeatedly.
